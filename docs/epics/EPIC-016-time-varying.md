# EPIC-016: Нестационарные очереди Mt/M/c (PSA / MOL)

- **Статус:** proposed
- **Создан:** 2026-07-11
- **Roadmap:** обзор трендов — [../research/queueing-trends-2026.md](../research/queueing-trends-2026.md) (вторая волна, п. C)

## Цель

Добавить анализ **нестационарных** систем с зависящей от времени интенсивностью входа λ(t)
(суточные/сезонные профили) — аппроксимации **PSA (pointwise stationary approximation)** и
**MOL (modified offered load)** для нестационарных вероятности блокировки и задержки.

## Контекст

Реальный трафик нестационарен (пики нагрузки), а большинство формул предполагают стационар. Классика
transient/нестационарного анализа:
- **PSA**: подставить λ(t) в стационарную формулу — точно при медленных изменениях и большой ёмкости.
- **MOL**: сгладить λ(t) откликом M/M/∞ (offered load `m(t)`), затем стационарная формула Erlang —
  лучше на быстрых изменениях; ключ для staffing.
- Применение: колл-центры, облачный autoscaling, ЦОД с суточными профилями.

Первоисточники: Green–Kolesar–Whitt (staffing нестационарных очередей), Massey–Whitt (Mt/M/c),
см. research.
