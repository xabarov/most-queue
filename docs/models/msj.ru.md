# Multiserver-job системы (MSJ)

[🇬🇧 English version](msj.md) · [← Каталог моделей](../models.ru.md)

![Схема multiserver-job](../figures/msj.ru.png)

**Простыми словами:** в современных ЦОД и GPU-кластерах одна заявка часто занимает **несколько
серверов одновременно** (ядра, GPU) на всё время работы — в отличие от классического M/M/c, где
заявка занимает один сервер. Заявка ждёт, пока освободится достаточно серверов сразу. Модель
«multiserver-job» — очень активная тема (среднее время отклика до недавнего времени было почти
неизвестно), и ни один другой open-source пакет её не даёт.

### FCFS MSJ — точное время отклика (малые системы)

**Описание:** k серверов; у каждого класса — **потребность в серверах** (сколько занимает
одновременно) и экспоненциальная интенсивность. FCFS с блокировкой головы очереди: старшие заявки
жадно заполняют серверы, первая не влезающая останавливает просмотр. Точное среднее время отклика
(общее и по классам) через CTMC над последовательностью заявок по порядку прихода — для малых k /
немногих классов / умеренной загрузки.

**Класс расчета:** `MsjExactCalc` (`most_queue.theory.msj`) ·
**Симулятор:** `MsjSim` (`most_queue.sim.msj`)

```python
from most_queue.theory.msj import MsjExactCalc, MsjClass

calc = MsjExactCalc(k=2, classes=[MsjClass(0.4, 1, 1.0), MsjClass(0.2, 2, 1.0)])
r = calc.run()          # r.v[0] общее среднее пребывание; r.v_per_class по классам
```

### Saturated MSJ — пропускная способность и порог устойчивости

**Описание:** Область устойчивости MSJ-системы характеризуется её *насыщенной* (всегда с бэклогом)
версией. `MsjSaturatedCalc` точно решает насыщенную CTMC и возвращает пропускную способность
`X_sat` — **максимальную суммарную интенсивность входа**, которую открытая система выдерживает: она
устойчива тогда и только тогда, когда Λ < X_sat. Воспроизводит product-form результаты устойчивости
Grosof, Harchol-Balter, Scheller-Wolf.

**Класс расчета:** `MsjSaturatedCalc` (`most_queue.theory.msj`)

```python
from most_queue.theory.msj import MsjSaturatedCalc, MsjClass

sat = MsjSaturatedCalc(k=2, classes=[MsjClass(1.0, 1, 1.0), MsjClass(1.0, 2, 1.0)])
x_sat = sat.run()       # макс. суммарная интенсивность входа (микс классов — из соотношения rates)
```
