# Prompt: MGnNegativeDisasterCalc — transitions on negative arrival (n=3, levels 0..4)

Use this prompt in a diffusion/image model (e.g., GPT Image / Nano-banana) to generate a transition diagram for `most_queue/theory/negative/mgn_disaster.py`.

---

Сгенерируй **векторную** диаграмму переходов для **DISASTER**-модели `MGnNegativeDisasterCalc` (n=3), **ярусы 0..4**.

## Стиль (как в серии)

- Фон: **белый или прозрачный**.
- Стиль: **flat vector**, аккуратный, но **красочный**.
- Цвет по ярусам (один ярус = один цвет): 0 голубой, 1 бирюзовый, 2 фиолетовый, 3 оранжевый, 4 розовый.
- Узлы: одинаковые rounded-rect, тонкий тёмный контур, светлая заливка.
- `D`-узлы: та же заливка яруса, но **пунктирный контур** и подпись **D**.
- Без заголовков сверху и без текста снизу.

## Компоновка (строго)

Сделай **5 горизонтальных строк** с метками слева: **0, 1, 2, 3, 4**.

- Строка 0: только узел **00**.
- Строка 1: слева узлы **10**, **01**, а **D** — строго **справа** на той же строке.
- Строка 2: слева узлы **20**, **11**, **02**, а **D** — строго **справа** на той же строке.
- Строка 3: слева узлы **30**, **21**, **12**, **03**, а **D** — строго **справа** на той же строке.
- Строка 4: слева узлы **30**, **21**, **12**, **03**, а **D** — строго **справа** на той же строке.

Расположение слева направо внутри строки: микросостояния по возрастанию \(b\) (например 30, 21, 12, 03), затем с большим отступом **узел D** справа.

## Стрелки (нарисуй ТОЛЬКО их)

### \(\delta\) (катастрофа) — строго в `D` на той же строке

Все \(\delta\)-стрелки должны быть **отдельными** (по одной от каждого микросостояния), направленными **вправо** и заканчиваться в `D` **той же строки**.
Если линии накладываются, используй **слегка изогнутые параллельные стрелки** к `D` (fan-in), но НЕ заменяй их цепочкой через другие состояния.

- Строка 1: 10 → D (δ), 01 → D (δ)
- Строка 2: 20 → D (δ), 11 → D (δ), 02 → D (δ)
- Строка 3: 30 → D (δ), 21 → D (δ), 12 → D (δ), 03 → D (δ)
- Строка 4: 30 → D (δ), 21 → D (δ), 12 → D (δ), 03 → D (δ)

### \(\gamma\) — только вверх по колонке `D`

- D(4) → D(3) (γ)
- D(3) → D(2) (γ)
- D(2) → D(1) (γ)
- D(1) → 00 (γ)

## Запреты (очень важно)

- **Никаких стрелок между микросостояниями** (вообще 0 стрелок между 10/01/20/11/02/30/21/12/03).
- **Никаких \(\delta\)-стрелок** куда-либо кроме `D` **той же строки**.
- Не рисуй \(\lambda_{+}\), \(\mu_1\), \(\mu_2\) и любые другие стрелки.
- **Нельзя заменять “несколько стрелок в D” на цепочку** вида 03→12→21→30→D. Такое запрещено: каждая стрелка должна идти **из микросостояния прямо в D**.
